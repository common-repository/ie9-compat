Compat Inspector is designed to help sites migrating to IE9-10-11 Standards Mode, not those running in legacy modes.
== Description ==
Compat Inspector is a JavaScript-based testing tool that analyzes your site while it runs.

Documentation can be found at
http://ie.microsoft.com/testdrive/HTML5/CompatInspector/help/post.htm
http://blogs.msdn.com/b/ie/archive/2011/04/27/ie9-compat-inspector.aspx
http://blogs.msdn.com/b/ie/archive/2012/01/20/ie10-compat-inspector.aspx

From 1 October 2016 on, this plugin is no longer supported. Please use [Powerpack](http://www.ramoonus.nl/wordpress/powerpack/) instead.

== Installation ==
1. Upload `ie9-compat.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
